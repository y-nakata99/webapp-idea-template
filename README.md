# Web App Idea Factory (Template)

このリポジトリは、アイディアを最小操作で Web アプリケーションとして形にするためのテンプレートです。

目的：
・アイディアを簡単・スピーディに Web アプリケーションにする
・PR ベースで安全に開発を進める
・codex を用いた実装を前提とする

前提：
・GitHub + Vercel を利用する
・main ブランチは常に本番として扱う

開発フロー概要：
1. アイディアを整理する
2. codex に実装を依頼する
3. Pull Request を確認して main にマージする
4. Vercel により自動デプロイされる

ブランチ運用：
・main：本番
・codex/<slug>：作業ブランチ
